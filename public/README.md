# Public Assets

Thả các file favicon/logo vào đây:

| File | Mục đích | Kích thước đề xuất |
|------|----------|-------------------|
| `favicon.ico` | Favicon trình duyệt | 32×32 hoặc multi-size |
| `favicon.png` | Favicon PNG (modern browsers) | 32×32 |
| `apple-touch-icon.png` | Icon cho iOS khi lưu trang | 180×180 |

## Cách tạo favicon nhanh

1. Chuẩn bị file logo gốc (PNG, kích thước lớn)
2. Truy cập https://favicon.io hoặc https://realfavicongenerator.net
3. Upload logo → Tải file được tạo sẵn
4. Copy `favicon.ico`, `favicon.png` (hoặc `favicon-32x32.png` đổi tên), `apple-touch-icon.png` vào folder này
